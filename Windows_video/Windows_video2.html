動画2 Windows 《スクリーンショット》

※本動画では キー と ボタン は同一の意味として扱います

説明

・キーボード右上にあるPrintScreen（PrtSc）ボタンを押すと、現在の画面を画像（スクリーンショット）として保存できます。
※この段階ではスクリーンショットは保存されていません。クリップボードに一時的に格納されています。（クリップボードはWindowsボタンとVボタンを同時に押すと表示できます）
・スクリーンショットを撮ってそのまま保存したい場合、WindowsキーとPrintScreen（PrtSc）ボタンを同時に押してください。
・画面の一部だけ、範囲を指定してスクリーンショットを撮りたい場合、WindowsキーとShiftキーとSボタンを押してください。
