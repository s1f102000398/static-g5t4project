動画１ Windows 《ウィンドウの整理, 拡大, 縮小》

※本動画では キー と ボタン は同一の意味として扱います

説明

・タブを選択してWindowsキーとキーボード右下にある方向キーを同時に押すと、指定した方向にタブを詰めることが出来ます。
※上下に詰めることもできます。手順としては、上の工程によってタブを詰めた状態で、そのタブを選択してWindowsキーと方向キーの上または下を同時に押してください。
・選択したタブを全画面表示にしたい場合は、Windowsキーと方向キーの上を同時に押してください。
・タブを小さくしたい場合は、Windowsキーと方向キーの下を同時に押してください
