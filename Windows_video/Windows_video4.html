動画4 Windows 《設定画面の表示, アプリの表示》

※本動画では キー と ボタン は同一の意味として扱います

説明

・キーボード左下にあるWindowsボタンを押すと、スタートメニュー（ピン留めしているアプリの表示・アプリの検索、アカウントのサインアウト・ユーザーの切り替え、電源の再起動・シャットダウン・スリープが出来る）が表示されます。
・WindowsボタンとIボタンを同時に押すと、設定画面を開くことが出来ます。
・Windowsボタンと数字キーを同時に押すと、画面下のタスクバーから該当するアプリを出すことが出来ます。 今回の場合、chromeが（スタートメニューとその右の検索タブを省いて）左から9番目にあるので、Windowsボタンと9のキーを同時に押すとchromeが表示されます。
